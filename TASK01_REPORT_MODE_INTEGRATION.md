# Task01 - 報告模式分數顯示集成完成

## 📊 功能說明

在**報告模式**中，點擊「開始報告」後，分組成員名單會自動加載并顯示每位學生的分數統計：

```
格式: 姓名 (學號)<組長>  (被點次數/舉手次數; 分數)

示例:
林俊雄 (414155259)<組長>  (1/3; 10)
謝孤諺 (413155020)        (2/4; 8)
陳書甄 (413155159)        (0/1; 0)
```

## 🎯 使用流程

### Step1: 進入報告模式（老師端）
1. 登入系統（老師身份）
2. 選擇「報告模式」
3. 設定：
   - 本堂課最高可得分 (預設15分)
   - 上課班別
   - 指定報告組

### Step2: 開始報告
- 點擊「開始報告」按鈕
- 系統自動：
  - 建立新課堂 Session
  - 跳轉到課堂互動頁面 (/sessions/{session_id})

### Step3: 查看分組成員分數
在課堂互動頁面，左側「分組成員」區域會自動加載所有成員及其分數：

```
👥 分組成員

┌─────────────────────────┐
│ 📁 第一組(L) 🎙️ 報告組 │
├─────────────────────────┤
│ 林俊雄 (414155259)<組長> │
│         (1/3; 10)      │
├─────────────────────────┤
│ 謝孤諺 (413155020)      │
│         (2/4; 8)       │
└─────────────────────────┘
```

## 🔄 實時更新

- 分數顯示實時更新（每2秒自動刷新）
- 當學生舉手、被點、或被評分時，分數立即刷新
- 無需手動刷新頁面

## 📝 顯示內容

每位成員顯示以下信息：

| 項目 | 說明 | 示例 |
|------|------|------|
| 姓名 | 學生全名 | 林俊雄 |
| 學號 | 學生學號 | 414155259 |
| 組長標籤 | 僅組長顯示 | <組長> |
| 被點次數 | status='A' 的次數 | 1 |
| 舉手次數 | 所有 hand_raises 記錄 | 3 |
| 評點分數 | ratings.star 的總和 | 10 |

## 🧪 測試步驟

### 情景 1: 查看初始分數（無活動）
1. 開始報告 → 進入 session 頁面
2. 驗證所有成員顯示 (0/0; 0)

### 情景 2: 舉手並被評分
1. 學生舉手
2. 老師點名並評分 (例如 5 星)
3. 驗證該學生分數更新為 (1/1; 5)

### 情景 3: 多次舉手
1. 同一學生舉手3次
2. 被點2次，評分分別為 4 星和 3 星
3. 驗證該學生分數顯示為 (2/3; 7)
